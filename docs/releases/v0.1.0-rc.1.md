# v0.1.0-rc.1 public seed

This release candidate is the first privacy-safe public seed of Agent Homelab.

## Included

- one inventory for combined, direct, and separate edge-gateway relay topologies
- Traefik 3.7.8 and Authelia 4.39.20 rendered Compose stacks
- paired Traefik middleware and Authelia policy for protected routes
- local and SSH plan/apply executors with staged validation and rollback
- bootstrap, service ingress, maintenance, OAuth relay, and troubleshooting skills
- generic and GOG/GWS OAuth adapter contracts
- synthetic fixtures, 25 tests, skill validation, artifact inspection, secret/privacy scanning, dependency audit, and container scanning

## Limitations

- Linux, Docker Compose, OpenSSH, Traefik file provider, and Authelia are the supported v1 stack.
- DNS-provider mutation, VPN installation, active SSH tunnels, Kubernetes, Nomad, and a web control plane are not included.
- Operators provide reachable node addresses, DNS, credentials, Authelia users, and encrypted state backups.
- Authenticated application behavior still requires operator verification after deployment.

Do not place real inventory, secrets, OAuth callback data, ACME state, or private network details in issues or commits.
